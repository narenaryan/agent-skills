# skill-creator

Slash command that turns a documentation URL (or a set of URLs) into one or more concise global skills, covering only the advanced / non-obvious knowledge.

## Command

- **[skill-creator](skill-creator.md)** — invoke with a doc URL to distill it into a skill under `~/.claude/skills/<name>/SKILL.md`.

## Install

```bash
./install.sh skill-creator
```
