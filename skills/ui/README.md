# ui skills

Practical UI and UX design patterns for navigation, content organization, forms, validation, accessibility, and interaction feedback.

## Skills

- **[ui-design-patterns](ui-design-patterns/SKILL.md)** - designing UI flows, navigation models, content structure, wizards, settings, workspaces, help systems, transitions, and visual hierarchy.
- **[ui-form-design](ui-form-design/SKILL.md)** - designing forms and inputs with effective controls, defaults, autocomplete, password guidance, validation, and error messaging.

## Install

```bash
./install.sh ui                 # install all
./install.sh ui-design-patterns # install one
```
